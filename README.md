# Neptune-Issue-tracker-and-Admin-Management-System
A PHP based web app combining security features and communication for issue reporting and admin functionality

# Neptune

Neptune is a security-focused issue tracking and admin management web application built using PHP and MySQL. The project was originally developed as part of university coursework but gradually expanded into a larger personal project focused on authentication, user management and secure workflow handling.

The overall goal of the system was to create something that felt more like a modern real-world platform rather than a basic university CRUD application.

---

## Features

- User registration and login system
- OTP verification
- Admin approval system
- Password reset request workflow
- Email reset request workflow
- Ticket / issue reporting system
- Audit and login logging
- Role-based access control
- Secure password handling
- Responsive custom UI

---

## Technologies Used

- PHP
- MySQL / MariaDB
- HTML / CSS
- JavaScript
- PHPMailer
- XAMPP
- Figma

---

## Security Focus

A large part of the project focused on implementing and experimenting with security concepts including:

- Secure authentication workflows
- Password policy enforcement
- OTP verification systems
- Session handling
- Audit logging
- Role separation
- Input validation
- Administrative oversight systems

---

## UI / Design

The frontend was custom designed around the "Neptune" theme using dark gradients, glassmorphism styling and a space-inspired aesthetic rather than relying on premade templates.

A lot of the UI planning and prototyping was initially done in Figma before implementation.

---

## Project Status

Still being improved and cleaned up over time. Future plans include improving backend structure, security hardening and potentially rebuilding parts of the system using a more modern framework stack.
